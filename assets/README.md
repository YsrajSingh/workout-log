# Muscle Images

Drop muscle group images here. Name them exactly as the `muscle_group` value in your notebook.

## Supported names:
- `chest.png`
- `back.png`
- `shoulders.png`
- `biceps.png`
- `triceps.png`
- `legs.png`
- `core.png`
- `cardio.png`

Any `.png`, `.jpg`, `.jpeg`, or `.webp` format works.
The notebook will auto-detect and display the image when you log that muscle group.
